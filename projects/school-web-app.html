School Management Web App

Overview

A PHP and MySQL-based web app for efficient school administration, featuring blogs, ID cards, assignments, user roles, categories, leads, registration and notifications.

Screenshots

Login Page

This web app features a secure login page where Admin, Teachers, and Students can log in using their email or username, and password. Each role is directed to a tailored dashboard, ensuring role-specific access and functionality.

Blog Page

This blog page is designed with full CRUD functionality, allowing users to create, edit, and delete blogs effortlessly. It includes options to add feature images, categorize posts, and customize header and footer scripts for a dynamic and engaging experience.

Lead Page

This lead management page displays all leads in a structured table format, enabling users to view detailed information for a single lead by clicking the 'View' button. Additionally, it includes an export feature to download all lead data as an Excel file for easy analysis.

Registration Page

The registration page allows the Admin to create new Teacher and Student accounts by filling out a comprehensive form with all necessary details. This streamlined process ensures accurate data entry and user creation.

I-Card Page

The I-Card management feature enables Admins to create, edit, and delete student I-Cards seamlessly. Students can access and print their personalized I-Cards directly from the platform for convenience.

Assignment Page

The Assignment module allows Admins and Teachers to create, edit, and assign assignments to specific classes. Students can view their assignments along with their status and submit their work. Once submitted, Teachers can review and grade the assignments, providing feedback as needed.

Profile Page

The Profile page is designed with two tabs for easy navigation. The first tab allows users to view and edit their personal details, while the second tab enables them to change their site password securely.
